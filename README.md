# Homebrew Tap for DotCor

Official Homebrew tap for [DotCor](https://github.com/justincordova/dotcor) - a symlink-based dotfile manager with Git integration.

## Installation

```bash
brew tap justincordova/dotcor
brew install dotcor
```

## Install from HEAD

To install the latest development version:

```bash
brew install --HEAD dotcor
```

## Usage

```bash
# Initialize DotCor
dotcor init

# Add a dotfile
dotcor add ~/.zshrc

# List managed files
dotcor list

# Check status
dotcor status

# Get help
dotcor --help
```
